Projects

COVID-19 Trend Analysis and Prediction

The COVID-19 Trend Analysis and Prediction project involved developing a comprehensive data analytics solution to analyze the global and Indian trends of COVID-19, including infection rates, recoveries, and death counts. By leveraging Pandas for data preprocessing and Plotly for creating interactive visualizations, the project provided a clear view of key metrics. Time series forecasting models were built using Facebook Prophet to predict COVID-19 cases for the upcoming week. The project offered actionable insights through visualizations of trends and predictions, enhancing the understanding of the pandemic's spread and aiding in decision-making for better preparedness and response.
